## Welcome to connect4
Play a game of connect4 against a computer enemy with varrying difficulties.
Get an overview of all features and learn how to play by using the help command (CLI) or using the help menu (GUI).

## How to play connect4 with CLI:

- open project in IntelliJ IDEA or another java IDE of your choice
- set ./src/connect4/view/Shell as main class
- execute main class

## How to play connect4 with swing GUI:

- open project in IntelliJ IDEa or another java IDE of your choice
- set ./src/connect4/view/SwingView/Main as main class
- execute main class
